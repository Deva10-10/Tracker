# 🎓 Student Academic & Skill Tracker

A smart web application designed to help students track their academic progress and skill development in one place.

This platform allows users to monitor subject-wise performance along with technical skills like Data Structures & Algorithms (DSA), Web Development, Coding, and Data Science. The goal is to improve productivity, maintain consistency, and visualize growth over time.

## 🚀 Features
- 📚 Track academic subjects (marks, progress, goals)
- 💻 Monitor skills like DSA, Web Dev, Coding, Data Science
- 📊 Visual dashboards for performance analysis
- 🎯 Set daily/weekly goals
- 📈 Progress tracking with graphs
- 🔐 User authentication (future scope)

## 🎯 Purpose
This project aims to help students stay organized, focused, and consistent in both academics and skill-building, which are essential for career growth in tech.

## 🛠️ Tech Stack (Planned)
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (or Python)
- Database: MongoDB

## 📌 Status
🚧 Currently in development (Phase 1)
